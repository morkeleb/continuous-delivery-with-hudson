Hudson place tous les fichiers de données dans ce répertoire du système de fichiers. Vous pouvez changer cette valeur :
  1. Soit en utilisant l'outil d'administration de votre conteneur web pour positionner la variable HUDSON_HOME.
  2. Soit en positionnant la variable HUDSON_HOME avant de lancer votre conteneur web.
  3. Soit (non recommandé) en modifiant le fichier web.xml dans hudson.war (ou sa version décompressée dans votre conteneur web).
Cette valeur ne peut être changée quand Hudson est lancé. Elle a principalement pour but de vous permettre de constater que votre configuration a bien été prise en compte.
